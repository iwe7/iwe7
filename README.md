### iwe7

> 高端 web 一站式解决方案

* angular6.0
* rxjs 响应式编程
* underscore 工具库

> 体积
> vendor 525Kb gzip后 128kb
> 包含underscore, rxjs, @angular/core, @angular/router, @angular/common, @angular/platform-browser, @angular/animations, @angular/forms 其他库均采用懒加载方式。
